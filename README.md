# odin-recipes
This project is a website about italien cuisine recipes where i applied 
the basics of html like using the anchor element,images, ordered lists and
unordered lists on webpages. 

this basic website consists of the main webpage that have anchor elements
to three diffrent webpages and each webpage have an image and a breif description
about a specific recipe after that there are the ingredients which are represented
in an unordered list and finaly the steps of the reicpe are represented in an ordered
list.
